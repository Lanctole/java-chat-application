# Java Chat Application

## Описание
Многопользовательское чат-приложение на Java с клиент-серверной архитектурой и графическим интерфейсом.

## Технологии
- Java 17
- Gradle (мультимодульная структура)
- Sockets (TCP/IP)
- Swing (GUI)
- Многопоточность (ExecutorService)

## Функционал
- Одновременное подключение нескольких клиентов
- Синхронизация сообщений между пользователями
- Автоматический перенос длинных строк
- Управление списком активных клиентов
